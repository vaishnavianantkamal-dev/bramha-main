🔧 TROUBLESHOOTING

Issue: "Cannot find module 'react-router-dom'"
Solution:
  $ cd brahmavalley-main
  $ rm -r node_modules
  $ npm install
  $ npm run dev

Issue: "VITE error"
Solution:
  $ cd brahmavalley-main
  $ rm -r .vite dist
  $ npm run dev

Issue: "Cannot connect to database"
Solution:
  1. Start XAMPP MySQL
  2. Check backend/.env credentials
  3. Verify database exists

Issue: "API returns 404"
Solution:
  1. Start XAMPP Apache
  2. Check API file exists
  3. Verify URL is correct

Issue: "CORS error in console"
Solution:
  1. Check backend/.env CORS_ORIGIN
  2. Restart Apache
  3. Clear browser cache
