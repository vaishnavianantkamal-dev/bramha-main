⚡ QUICK START (5 MINUTES):

1. Start XAMPP
   └─ Open XAMPP Control Panel
   └─ Click Start for Apache
   └─ Click Start for MySQL

2. Create Database
   └─ Go to: http://localhost/phpmyadmin
   └─ Click New → Enter "brahmavalley_db" → Create
   └─ Click Import → Choose "backend/database/ALL_TABLES_COMPLETE.sql"

3. Start Frontend
   └─ Open Terminal
   └─ cd brahmavalley-main
   └─ npm install
   └─ npm run dev

4. Open Browser
   └─ Go to: http://localhost:5174

5. If White Screen
   └─ Go to: http://localhost:5174/diagnostic
